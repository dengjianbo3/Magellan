# AI 投资报告 Agent - V3 一级市场投研工作台 设计文档 (v1.1)

## 1. 核心范式转移：从“二级市场分析”到“一级市场尽职调查(DD)与投资备忘录(IM)辅助”

V3版本的核心目标，是把产品从一个面向二级市场投资者的分析工具，**转型**为一个服务于一级市场（私募股权/风险投资）投研团队的**专业工作台**。

我们的用户不再是个人投资者，而是**投资负责人**和**投资决策委员会 (IDC)**。他们最关心的问题不是“股票能否上涨”，而是：

1.  **流程管理：** 如何高效、规范地完成一个项目的尽职调查（DD）？
2.  **信息整理：** 如何从海量的非结构化材料（BP、访谈纪要）中提取有效信息？
3.  **报告撰写：** 如何快速生成一份逻辑严谨、要素齐全的投资备忘录（IM）用于上会？

因此，V2的核心理念——**“秩序感与透明度”、“逻辑严谨与可控性”、“完整性与交互性”**——将被继承并强化，以完美满足一级市场的专业需求。

---

## 2. Agent团队与工具升级：适配一级市场数据与逻辑

V3将对V2的Agent团队和他们使用的工具（微服务）进行一次“换核”手术。

### a. 数据摄取层 (Data Ingestion Layer)

*   **智能文档解析器 (Smart Parsers):**
    *   **V3升级:** `pdf_parser`, `excel_parser`, `word_parser` 将从简单的文本提取，升级为基于多模态模型（如 Gemini Pro Vision）的**布局感知智能解析**。
    *   **核心能力:** 能够识别文档结构（如团队介绍、财务预测表），并提取结构化字段（如创始人背景、市场规模TAM、融资金额、估值等）。

*   **`external_data_service` (外部数据服务):**
    *   **V3重构:** `public_data_service` 将被此服务取代。
    *   **核心能力:** 对接一级市场专业数据库（如 Crunchbase, PitchBook, 天眼查, IT桔子），获取工商信息、融资历史、竞品对比。并集成领英等渠道，进行团队背景调查。

*   **`web_search_service` (实时网络搜索服务):**
    *   **V3新增:** 基于Tavily等为LLM优化的搜索API。
    *   **核心能力:** 为尽职调查提供实时的、广泛的网络信息输入，用于交叉验证BP中的信息、进行深度团队背景调查和风险发现。

*   **`internal_knowledge_service` (内部知识库服务):**
    *   **V3新增:** 这是V3的战略级新服务，将使用**向量数据库**实现。
    *   **核心能力:** 存储和检索公司内部所有的历史项目资料、投资备忘录、行业研究和专家纪要。在分析新项目时，Agent能自动查询“我们内部是否看过相似项目？”

*   **`openbb_service` (金融数据平台服务):**
    *   **V3新增 (高级功能):** 封装OpenBB开源金融数据平台。
    *   **核心能力:** 主要用于投研后期，提供**可比上市公司分析 (Public Comps)** 和 **退出路径分析 (Exit Analysis)** 所需的公开市场数据和估值倍数。

### b. 智能体逻辑层 (Agent Logic Layer)

*   **`OrchestratorAgent` (总控Agent):**
    *   **V3工作流:** 其状态机将从二级市场分析流程，重定义为一级市场的**尽职调查 (DD) 标准流程**，例如：`文档解析 -> 团队背景调查 (TDD) -> 市场与竞品分析 (MDD) -> 交叉验证 -> 生成DD问题清单`。

*   **`AnalysisAgent` (分析Agent):**
    *   **V3进化:** 这是V3的核心进化。它将不再生成通用公司简介，而是生成VC分析的**核心模块**：
        1.  **团队分析 (Team):** 综合背景信息，评估团队经验匹配度和执行力。
        2.  **市场分析 (Market):** 交叉验证BP中的市场规模和增长假设。
        3.  **产品与技术 (Product):** 分析技术壁垒和护城河。
        4.  **竞品格局 (Competition):** 自动生成竞品象限图或功能对比表。
        5.  **财务与估值 (Financials & Valuation):** (高级阶段) 调用`openbb_service`进行可比公司分析。

*   **`RiskAgent` (风险Agent):**
    *   **V3进化:** 其产出从“关键追问问题”升级为V3的杀手级功能——**面向创始人的DD问题清单**。例如：“BP第10页提到月活增长30%，请提供详细的同期群数据和获客成本。”

*   **`user_service` (用户服务):**
    *   **V3升级:** 从“个人投资风格”扩展为**“机构投资偏好”**。
    *   **核心能力:** 允许机构设置自身的“投资主题 (Thesis)”、“偏好阶段 (Seed/A/B轮)”、“关注赛道 (SaaS/AI)”等。Agent在分析之初即可判断项目与机构的匹配度。

## 3. UI/UX重塑：从“交互式报告”到“IM撰写工作台”

V3的最终交付物将从一份“只读”的分析报告，演变为一个支持决策过程的**“投资备忘录 (IM) 撰写工作台”**。

*   **品牌色更新:** 整个UI界面的主色调将从紫色系更新为**橘红色系** (例如 `#FF4500` OrangeRed)，以传递更具活力、果断和专业的视觉感受。

*   **`ChatView.vue` (The Foundry):**
    *   **V3定位:** **完美复用**。其“类CI/CD”的界面非常适合展示漫长而复杂的DD流程，能给予投资负责人极大的掌控感和秩序感。

*   **`InteractiveReportView.vue` (交互式报告):**
    *   **V3重塑:** 将其重塑为一个**“IM草稿生成器”**，并延续三栏式布局。
    *   **左栏 (大纲):** 章节更新为标准IM格式（投资亮点、团队、产品、市场、风险等）。
    *   **中栏 (内容区):**
        *   **核心升级:** 变为一个**可编辑的富文本区域**。
        *   Agent生成的各模块分析将自动填入对应章节，投资负责人可在此基础上进行修改、补充自己的定性判断。
    *   **右栏 (Agent助手):**
        *   保留“核心DD问题清单”。
        *   新增**“内部洞察”**模块，由`internal_knowledge_service`驱动，提示“合伙人A曾在2023年评论过此赛道...”。
        *   新增**“一键导出”**功能，将中栏内容导出为Word或PDF。

*   **人机交互节点 (HITL) 优化:**
    *   **节点1 (歧义消除):** 保留，依然重要。
    *   **节点2 (深度分析):** 重新定义为**“添加您的访谈纪要”**。AI完成初步IM草稿后，投资负责人与创始人访谈，然后回到系统，上传会议纪要或输入自己的判断。Agent接收新信息后，将自动更新IM草稿和风险清单。
