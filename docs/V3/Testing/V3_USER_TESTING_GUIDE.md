## 📋 测试前准备

### 1. 环境检查

在开始测试前，请确认：

```bash
# 1. 确认所有后端服务正常运行
docker ps | grep magellan

# 应该看到 8 个容器在运行：
# - magellan-report_orchestrator
# - magellan-llm_gateway-1  
# - magellan-web_search_service
# - magellan-user_service-1
# - magellan-internal_knowledge_service
# - magellan-external_data_service
# - magellan-chroma-1
# - magellan-excel_parser-1
```

**✅ 检查点 1**: 所有容器状态为 `Up`
- [ ] 8个容器全部运行
- [ ] 无异常退出

---

### 2. 启动前端

```bash
cd frontend
npm run dev
```

**✅ 检查点 2**: 前端启动成功
- [ ] 控制台显示 `Local: http://localhost:5173/`
- [ ] 无编译错误
