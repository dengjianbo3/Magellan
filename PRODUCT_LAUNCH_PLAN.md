## 八、风险与应对

| 风险 | 影响 | 应对措施 |
|------|------|----------|
| LLM API 不稳定 | 分析失败 | 重试机制 + 降级方案 |
| WebSocket 断连 | 进度丢失 | 持久化 + 自动重连 |
| 高并发压力 | 服务崩溃 | Kafka 削峰 + 限流 |
| 数据安全 | 泄露风险 | 加密 + 权限控制 |
